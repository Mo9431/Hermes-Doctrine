# Hermes Doctrine

Hermes is a structural heuristic compiler and intelligence pipeline. It applies Recursive Trajectory Compression (RE-TRAC) to translate noisy, unstructured operational narratives—such as cyberattacks, fraud, or systemic anomalies—into pure, deterministic structural capital.

## The Pipeline

1. **Dialectic Engine (Hermes Architect)**
   The human-in-the-loop chat interface designed for high-density, analytical communication, operating functionally as the primary blueprint architect.

2. **RE-TRAC (Recursive Trajectory Compression)**
   Takes noisy logs or incident variance and algebraically distills constraints into a dense First-Order Logic (FOL) formulation and strict Symbol Dictionary.

3. **Abstraction Router**
   A classification gateway that evaluates the incoming mathematical dictionary against established Level-1 structural unifying abstractions (e.g., `Sybil_Aggregation_Rotation`), dynamically spinning up visual instances in the persistence system.

4. **Vulcan Synthesis Engine**
   Compiles FOL definitions directly into robust backend algorithms. Vulcan adheres strictly to a Target Binding Contract configured to output real-time streaming implementations (Node/Redis) or batch analytic scripts (Python/Pandas). Synthesis includes hard constraints preventing unscalable implementations like nested loops to enforce $O(E \\log E)$ graph theory strategies or Vectorized efficiency.

5. **The Validation Gauntlet**
   Standalone validation endpoints designed to execute "Live-Fire" replay stress-testing of generated scripts securely against chronological spoof events.

6. **The Skill Grimoire**
   A React-built UI dashboard that visualizes the FTS5 intelligence library. It stores and enumerates abstract algorithms passing down to Level-2 verified implementations.

## Domains Covered

* **Crypto_Sybil_Swarm** - Identifies narrow-time sub-threshold deposit convergences to a singular sink.
* **Ticketmaster_Cart_Hold** - Detects dynamic daisy-chained checkout timers maintaining perpetual inventory lock.
* **AML_Smurf_Structuring** - Discovers coordinated money-mule activity bypassing the $10k reporting limit using mathematical bijectivity (Python/Pandas).
* **Ad_Fraud_Sybil_Click** - Identifies programmatic click-fraud operating inside specific chronological bounds utilizing diverse network infrastructure.

## Stack & Architecture

* **Frontend**: React (Vite), Tailwind CSS, Lucide Icons.
* **Backend**: Express on Node.js (via `tsx`).
* **AI Orchestration**: Built-in Multi-LLM Cascade Gateway orchestrating failovers sequentially across specialized frontier models (Arcee, xAI, Anthropic, OpenAI, OpenRouter) and strict structural enforcements leveraging Zod.

## Setup & Execution

### Install Dependencies
```bash
npm install
```

### Environment Configuration
Copy the exemplar environment list and provide the keys for the multi-provider routing cascade.

```bash
cp .env.example .env
```

### Launch Development Server

```bash
npm run dev
```

The system requires `tsx` to serve the backend router endpoints paired with Vite's middleware serving the React interface.
